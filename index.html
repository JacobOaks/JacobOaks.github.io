Projects



GeoArt

GeoArt is a computational art program that my friend and I made for SteelHacks 2019. It is an iOS app that uses the location of the user to create a piece of art entirely unique to that location. We only had 24 hours, so the art wasn't very beautiful, but it was interesting to see the differences between places far apart. Ambulare

Ambulare is a solo project that I have worked on to teach myself graphics rendering using OpenGL. It is a simple side-scrolling videeo game with a custom physics engine, rendering framework, shader programs, and set of graphics - all built upon the light-weight java game library. While Ambulare never got to a point where it had a lot of gameplay to offer, I learned a lot about design, user experience, rendering, and physics.
